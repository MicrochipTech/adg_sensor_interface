## Sensor interface with LX7730 Example Summary

This example implements an interface that run on SAMRH71F20-EK board to connect the LX7730 device to a computer GUI. This firmware convert GUI commands received on the UART link to SPI commands to read and write LX7730 registers. Sensors are connected to the LX7730 daugther board ADC channels.
